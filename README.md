## 簡単な使い方
 ![実行例](./robosys.png)
        ↑実行しているターミナルの様子
* 1.ファイルの入っているディレクトリに移動
* 2.整数を出力する
* 3.出力した整数をplusが受け取る
* 4.実行。数字の足された値がターミナルに出力される。
