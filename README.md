# < plusコマンド>
![test](https://github.com/gaku-3319/robosys202x/actions/workflows/test.yml/badge.svg)

標準入力から読み込んだ数字を足すコマンド
## 簡単な使い方
* １． OSにplusのファイルをダウンロードする。 
* ２．./plusと入力し、実行する。
* ３．数値を入力することが出来るようになるので、入力しenter

## 必要なソフトウェア

* Python
* テスト済みバージョン: 3.7～3.10

## テスト環境

* Ubuntu18.04
